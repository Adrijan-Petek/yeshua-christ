# Yeshua-Christ

A faith-focused mini app for sharing the Gospel freely.

## Features

- **Faith & Testimony**: Share on Farcaster with Christ-centered content.
- **Videos**: Embed YouTube videos for sermons, worship, etc.
- **Bible**: Free downloads and online reading with verse sharing.

## Getting Started

1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- next-themes for dark/light mode

## Principles

This app exists only to glorify Christ and spread the Gospel freely. No hype, no speculation, no monetization.
